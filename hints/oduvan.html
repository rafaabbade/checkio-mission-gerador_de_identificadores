Como combinar strings e variáveis de forma eficiente?

Você pode usar f-strings ou concatenação para combinar strings e variáveis de forma eficiente. 

cidade = "São Paulo"
estado = "SP"
cep = 12345
# f-string
endereco_id = f"{cidade}-{estado}-{cep}"
# concatenação
endereco_id = (cidade + "-" + estado + "-" + cep)
