Enunciado

Tina, desenvolvedora de um grande sistema de gerenciamento de conteúdo, precisa criar identificadores únicos para cada novo usuário que se registra. Crie uma função que combine o primeiro nome, o sobrenome e o ano de nascimento do usuário para formar um identificador único. O identificador deve ser todo em letras minúsculas e as partes devem ser separadas por um hífen.

Entrada: Duas strings, primeiro_nome, sobrenome e um inteiro, ano_nascimento.

Saída: Uma string representando o identificador único.
